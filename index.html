About

Portal World is a Portal 2 streaming bot inspired by Warp World. Like Warp World, the tool allows viewers to submit levels for the streamer to play. The difference is that this application was made for Portal 2 custom levels instead of Super Mario Maker levels. However it does work with all steam workshop items and is thus not limited to only Portal 2 levels, the aesthetics are tooled towards that game though.

This tool is an expansion of an old, extremely simplistic, system I made to display viewer levels quickly on stream, it worked through the streamer manually filling in the level and author name with no added functionality. I kept adding on to that till I had this tool.

Features

  • Level submission through Steam Workshop links
  • Automatic level author detection through Steam
  • A Portal themed bar displaying level information on stream
  • A browser based dashboard with level queue
  • Possibility to lock the queue preventing people from adding more levels to the queue

Tutorial

Initial setup

After downloading and installing the program in an easy to reach spot, go to its files and into the "dir" folder. In there open settings.txt

Fill in all fields correctly, keep in mind that the 'NAME' setting refers to the username of the account used to gather the oauth token while 'CHANNEL' refers to the name of the channel the bot will be active on. And that the help command is set to 'commands' by default.

Click here to display image

In this example the bot will be active on Knipje's channel using their own account, allow every user to submit up to 3 levels at any given moment, listen to commands starting with ! and has a help command set to commands (so !commands in chat). Also the queue command will cap off at showing 10 levels to prevent chat clogging.

After you have filled in the settings entirely you can launch Portal_World.exe (or any shortcuts you may have made during installation) and, if all went well, a page should open in your browser tell you there are no levels in the queue, if this page does not show up please check if you filled everything in correctly or submit a bug report here.

Setting up the stream bar

Setting up the stream bar is relatively easy, you will need to have the program running in order to make it work. Firstly add a new browser source leading to localhost:8000, then fill in the correct width of your stream (most likely 1980) and crop and move the bar as needed.

Optionally, you could also set your personal queue screen up to be displayed on stream in a similar manner, to do this follow all steps above but have the browser source lead to localhost:8000/queue.html instead.

Chat commands

The proper commands (by default) for the bot are as follows:

Command Variable Aliases Function
!add [Steam Workshop link] !submit Adds a level to the queue
!remove [Level name or url] !delete Removes a level to the queue
!promote [Level name or url] Promotes a level be the next level in the queue, mod only command
!list None !queue, !q Lists all levels
!mylist None !myqueue, !myq Lists all levels submitted by the person who used the command
!current None !currentlevel, !nowplaying, !now, !np Displays the current level
!clear None !reset, !empty Resets the list of levels, mod only command
!lock None Locks the queue, mod only command
!unlock None Unlocks the queue, mod only command
!next None !skip Removes first level from queue, mod only command
!commands None None Displays all all-users commands optional

General usage

After the bot has started up normally, viewers will be able to use all commands mentioned above. Remember that the account used for the bot cannot use commands itself. The streamer should then click the "Next Level" button to have the next level appear on screen, open the link in your browser and subscribe to the workshop item, it should be downloaded automatically. The queue should persist between uses of the bot, if you want to clear the queue either use the corresponding button or command.
